# Advent of Code Sketchbook

## Prep for new day
### Rust
copy day-00 directory and paste as day-xx where xx is the current day
update day-00 in the following files:
```bash
Cargo.toml
src/bin/part1.rs
src/bin/part2.rs
```

### Setup cargo watch
while in the day directory:
```bash
cargo watch -x "run --bin part-x"
```
